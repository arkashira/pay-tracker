# breakeven.md

## Unit Economics & Break-even Analysis

### Cost per Active User
- **Compute**: $0.10/user/month
- **Storage**: $0.05/user/month
- **Bandwidth**: $0.02/user/month
- **Total Cost per Active User**: 
  - **Compute**: $0.10
  - **Storage**: $0.05
  - **Bandwidth**: $0.02
  - **Total**: **$0.17/user/month**

### Pricing Tiers
| Tier Name           | Monthly Price ($) | Features                                                                 |
|---------------------|-------------------|--------------------------------------------------------------------------|
| **Basic**           | $15               | Basic invoice tracking, payment status updates, email notifications      |
| **Pro**             | $30               | All Basic features + advanced filtering, reporting tools, API access     |
| **Enterprise**      | $60               | All Pro features + dedicated support, custom integrations, priority updates |

### Customer Acquisition Cost (CAC) Range
- Estimated CAC: **$50 - $100** per user

### Lifetime Value (LTV) Estimate
- Average Monthly Revenue per User (ARPU) for Pro Tier: $30
- Average Customer Lifespan: 24 months
- LTV Calculation: 
  - LTV = ARPU × Average Customer Lifespan
  - LTV = $30 × 24 = **$720**

### Break-even Users Count
- Fixed Costs (e.g., marketing, salaries, infrastructure): **$5,000/month**
- Contribution Margin per User (using Pro Tier pricing): 
  - Revenue per Pro User: $30
  - Cost per User: $0.17
  - Contribution Margin = Revenue - Cost = $30 - $0.17 = **$29.83**
- Break-even Users Count Calculation:
  - Break-even Users = Fixed Costs / Contribution Margin
  - Break-even Users = $5,000 / $29.83 ≈ **168 users**

### Path to $10K MRR
- Target Monthly Recurring Revenue (MRR): **$10,000**
- Pricing Tier Focus: Pro Tier ($30/month)
- Users Needed for $10K MRR:
  - Users = Target MRR / Price per User
  - Users = $10,000 / $30 ≈ **334 users**
  
### Summary
To achieve $10K MRR, we should focus on acquiring approximately **334 Pro Tier users**. Given the CAC range of $50 - $100, the initial investment in user acquisition would be between **$16,700 - $33,400**. The break-even point is at **168 users**, which indicates a healthy margin for scaling beyond the break-even threshold.
