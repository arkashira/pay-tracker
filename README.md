# Pay Tracker

A simple pay tracker application that sends reminders for partial or overdue invoices.

## Usage

1. Create an instance of the `PayTracker` class.
2. Add invoices using the `add_invoice` method.
3. Add email templates using the `add_email_template` method.
4. Send reminders using the `send_reminders` method.
5. Log emails using the `log_email` method.
6. Get sent emails using the `get_sent_emails` method.
7. Edit email templates using the `edit_email_template` method.
8. Get email templates using the `get_email_template` method.
