# Pay Tracker
A simple pay tracker application.

## Usage
1. Create an instance of the `PayTracker` class.
2. Add invoices using the `add_invoice` method.
3. Prioritize invoices using the `prioritize_invoices` method.
4. Save the prioritized list using the `save_prioritized_list` method.
5. Load the prioritized list using the `load_prioritized_list` method.
6. Send notifications for overdue invoices using the `send_notifications` method.
